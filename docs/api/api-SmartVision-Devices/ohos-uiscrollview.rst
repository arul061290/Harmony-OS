OHOS::UIScrollView
==================

**Overview**\ 
--------------

**Related Modules:**

`Graphic <graphic.rst>`__

**Description:**

Supports horizontal or vertical scroll of child views. This class is
inherited from `UIAbstractScroll <ohos-uiabstractscroll.rst>`__.

Horizontal or vertical scroll occurs only when the width or height of
the child view is greater than that of the
`UIScrollView <ohos-uiscrollview.rst>`__.

**Since:**

1.0

**Version:**

1.0
