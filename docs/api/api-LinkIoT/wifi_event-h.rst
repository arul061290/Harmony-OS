wifi_event.h
============

**Overview**\ 
--------------

**Related Modules:**

`Wifiservice <wifiservice.rst>`__

**Description:**

Defines callbacks and structure of Wi-Fi events.

`RegisterWifiEvent <wifiservice.rst#ga0e4d452915d1588664b91837b821eada>`__
can be used to listen for Wi-Fi connection, disconnection, and scan
events.

1.0
