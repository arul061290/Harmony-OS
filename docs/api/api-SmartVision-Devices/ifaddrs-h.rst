ifaddrs.h
=========

**Overview**\ 
--------------

**Related Modules:**

`NET <net.rst>`__

**Description:**

Provides functions and data structures related to network operations.

For example, you can use the functions to send and receive network data,
manage network addresses, and convert bytes.
