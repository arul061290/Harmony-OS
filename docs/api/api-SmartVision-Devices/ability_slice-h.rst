ability_slice.h
===============

**Overview**\ 
--------------

**Related Modules:**

`AbilityKit <abilitykit.rst>`__

**Description:**

Declares ability slice-related functions, including ability slice
lifecycle callbacks and functions for connecting to or disconnecting
from ability slices.

**AbilitySlice** instances, which are specific to
`Feature <feature.rst>`__ Abilities (abilities using the Page template),
are used to present different screens on an applicationâ€™s user
interface. A `Feature <feature.rst>`__ Ability can have multiple ability
slices.
