ui_abstract_scroll.h
====================

**Overview**\ 
--------------

**Related Modules:**

`Graphic <graphic.rst>`__

**Description:**

Declares the base class used to define the attributes of a scroll. The
**UIList**, **UIScrollView**, and **UISwipeView** inherit from this
class.
