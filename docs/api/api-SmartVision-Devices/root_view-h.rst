root_view.h
===========

**Overview**\ 
--------------

**Related Modules:**

`Graphic <graphic.rst>`__

**Description:**

Manages a root view.

A root view is the view containing its child views. It represents the
root node in a tree structure and is the parent to all the children.

1.0
