OHOS::UIToggleButton
====================

**Overview**\ 
--------------

**Related Modules:**

`Graphic <graphic.rst>`__

**Description:**

Represents a toggle button.

Each toggle button contains two images, one for pressing and the other
for releasing.

1.0
