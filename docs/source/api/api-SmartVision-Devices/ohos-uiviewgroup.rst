OHOS::UIViewGroup
=================

**Overview**\ 
--------------

**Related Modules:**

`Graphic <graphic.md>`__

**Description:**

Represents a view group that consists of its child views.

The child views can be added to, inserted in, and removed from the view
group. Child views that are added later are displayed at the upper layer
of this view group. All child views are stored in a linked list.

**Since:**

1.0

**Version:**

1.0
