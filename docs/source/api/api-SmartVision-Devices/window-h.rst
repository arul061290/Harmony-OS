window.h
========

**Overview**\ 
--------------

**Related Modules:**

`Window <window.md>`__

**Description:**

Declares the **Window** class that provides a drawing canvas for the
**RootView**, which represents the root node of a view tree.

Each window is bound to a **RootView**. For details, see **RootView**.
The **Window** class also provides window management capabilities,
including creating, destroying, showing, hiding, moving, resizing a
window, raising a window to the top, and lowering a window to the
bottom.

1.0

1.0
