OHOS::UIDialog
==============

**Overview**\ 
--------------

**Related Modules:**

`Graphic <graphic.rst>`__

**Description:**

Represents a dialog box.

A dialog box displays information for user input. After a button is
clicked in the dialog box, the corresponding callback event is
triggered.

**Since:**

1.0

**Version:**

1.0
