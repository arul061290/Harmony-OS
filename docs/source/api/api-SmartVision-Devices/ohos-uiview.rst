OHOS::UIView
============

**Overview**\ 
--------------

**Related Modules:**

`Graphic <graphic.md>`__

**Description:**

Defines the base class of a view, providing basic view attributes and
operations. All views are derived from this class.

**Since:**

1.0

**Version:**

1.0
