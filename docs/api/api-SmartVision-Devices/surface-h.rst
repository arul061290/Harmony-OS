surface.h
=========

**Overview**\ 
--------------

**Related Modules:**

`Surface <surface.md>`__

**Description:**

Provides the capabilities of applying for and releasing shared memory in
multimedia and graphics scenarios.

By default, a surface can apply for a shared buffer. You can call
**SetQueueSize** to set the maximum number of buffers that can be
applied for.

1.0
