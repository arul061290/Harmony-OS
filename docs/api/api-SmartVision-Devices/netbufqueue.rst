NetBufQueue
===========

**Overview**\ 
--------------

**Related Modules:**

`WLAN <wlan.rst>`__

**Description:**

Indicates the queues for storing network data.

Queues can be used to process multiple pieces of network data in a
centralized manner, improving efficiency.

1.0
