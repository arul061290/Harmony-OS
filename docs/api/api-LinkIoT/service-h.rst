service.h
=========

**Overview**\ 
--------------

**Related Modules:**

`Samgr <samgr.rst>`__

**Description:**

Provides basic types and constants of services.

This file is mainly used for service development. This file provides
basic capabilities such as lifecycle functions of services, inherited
macros, and task configuration.

1.0
