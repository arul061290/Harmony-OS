DevHandle
=========

**Overview**\ 
--------------

**Related Modules:**

`COMMON <common.rst>`__

**Description:**

Defines the common device handle of the platform driver.

The handle is associated with a specific platform device and is used as
the first input parameter for all APIs of the platform driver.

**Since:**

1.0
