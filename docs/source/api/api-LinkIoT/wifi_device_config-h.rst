wifi_device_config.h
====================

**Overview**\ 
--------------

**Related Modules:**

`Wifiservice <wifiservice.md>`__

**Description:**

Defines the Wi-Fi station configuration.

The Wi-Fi station configuration includes the security type and data
length.
